Tutorial

Lession 0

Press the plus and minus buttons in the navigation bar (next to the Tutorial button) and see what it means for the number of columns on the screen. (If using a very thin screen, click the minus button while on this tutorial page to get only one column on the screen.) Try to open some columns by clicking on the various links in the columns. Try to open more columns than can fit inside the screen at once and then cycle through them using either the left and right arrow keys or by clicking the margins of the page (with the arrow symbols in them).

Lession 1

Once this is done, refresh the page to get only one column called "Entities." This is the fundamental entity in the database which represents the category of all entities in the database.
